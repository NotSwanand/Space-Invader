🚀 Space Invader – Pygame Edition 👾  

A classic **Space Invaders** game built using **Python** and **Pygame**. Control your spaceship, shoot bullets, and eliminate enemy UFOs while avoiding their descent.  

![Space Invader Screenshot](screenshot.png) *(Add a screenshot of your game here)*  

---

📌 Features  
✅ Smooth player movement (left & right)  
✅ Multiple enemies with increasing difficulty  
✅ Bullet shooting mechanics  
✅ Collision detection  
✅ Score tracking system  
✅ Background music and sound effects  
✅ Game Over screen  

---

🎮 Controls  
🔼 **Spacebar** – Shoot bullets  
⬅️ **Left Arrow** – Move left  
➡️ **Right Arrow** – Move right  

---

🔧 Installation & Running the Game  

1️⃣ Clone the Repository  
```sh
git clone https://github.com/NotSwanand/Space-Invader.git
cd Space-Invader
```

2️⃣ Install Dependencies  
Make sure you have **Python 3.x** installed. Then, install **Pygame**:  
```sh
pip install pygame
```

3️⃣ Run the Game  
```sh
python space_invaders.py
```

---

⚙ Requirements  
- Python 3.x  
- Pygame  

---

🛠 Customization  
Want to modify the game?  
- You can change **enemy speed, bullet speed, number of enemies**, and more in the script.
